# Contributing

Kaze Rwanda is a private business project and is not open source.

Contributions are not accepted publicly. Work on this repository is limited to the owner, the internal team, paid collaborators, or people working under a separate written agreement.

## Internal Contribution Rules

- Keep the Kaze brand premium, clear, and easy to understand.
- Do not add generic template content or placeholder copy to production-facing pages.
- Do not commit secrets, tokens, private keys, customer data, or deployment credentials.
- Run `npm run build` before handing off website changes.
- Keep ownership, legal, and contact information accurate.
