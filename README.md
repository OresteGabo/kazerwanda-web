# Kaze Rwanda Website

Official website for Kaze Rwanda, a hospitality and event platform by GABO.

Official website: [https://kazerwanda.com](https://kazerwanda.com)

Canonical brand destination: `kazerwanda.com` is the main public home for Kaze Rwanda. This GitHub repository supports the website and should reinforce the same product identity, not replace the website as the primary brand page.

Kaze Rwanda presents events (weddings, conferences,...), Kaze Pass guest access, hotel maps, event services, local payment options, and partner onboarding for hotels, venues, and organizers in Rwanda.

## Tech Stack

- Vue 3
- Vite
- CSS modules through the app-level stylesheet
- Static website output in `dist/`

## Getting Started

Install dependencies:

```sh
npm install
```

Run the local development server:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```

Preview the production build:

```sh
npm run preview
```

## Project Structure

- `src/App.vue` contains the current landing page content and Vue data arrays.
- `src/assets/base.css` contains global tokens and base browser styles.
- `src/assets/main.css` contains the website layout, responsive design, and visual direction.
- `src/assets/` contains Kaze and GABO brand assets used by the page.
- `public/` contains static public files such as the favicon.

## Brand Notes

The website should feel premium, warm, and Rwandan-hospitality focused. Avoid generic startup templates and avoid heavy, unreadable visuals. Kaze should be the main product brand, with GABO shown as the creator/owner attribution.
