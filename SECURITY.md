# Security Policy

## Reporting Security Issues

If you find a security issue related to the Kaze Rwanda website or its infrastructure, contact:

[orestegabo@icloud.com](mailto:orestegabo@icloud.com)

Do not publicly disclose security issues before the owner has reviewed and resolved them.

## Sensitive Data

Do not commit secrets, tokens, credentials, private keys, analytics secrets, deployment keys, or customer data to this repository.

Use local environment files for secrets when needed, and ensure those files remain ignored by git.

## Current Status

This website is currently a static Vue/Vite website. Future integrations such as forms, analytics, payments, or dashboards must be reviewed for privacy and security before production use.
