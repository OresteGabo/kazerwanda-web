# Kaze Rwanda Website Roadmap

This roadmap tracks website work only. Product and mobile-app roadmap items belong in the main Kaze application repository.

## Done

- Created a Vue 3 + Vite landing page.
- Added Kaze and GABO logo assets.
- Added a first responsive visual system for the website.
- Added production build support through `npm run build`.

## Next

- Replace the current favicon with a Kaze-branded favicon.
- Add stronger mobile polish after testing on real device sizes.
- Add a dedicated partner/contact form when backend or form handling is ready.
- Add legal pages or links for privacy policy, terms, ownership, and contact.
- Add SEO metadata and social sharing preview tags in `index.html`.
- Add real screenshots or carefully designed product mockups when the mobile app stabilizes.
- Add analytics only after privacy requirements are clear.

## Later

- Add pages for venue owners, service providers, investors, and product documentation.
- Add a press/media kit with approved logos and brand usage rules.
- Add localized content for English, French, and Kinyarwanda when translations are ready.
- Add CMS or API-backed content if the website needs frequent updates.
- Add deployment automation for the selected hosting provider.
